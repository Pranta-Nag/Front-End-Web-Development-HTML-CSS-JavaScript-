📄 HTML5 Demo Page
📌 Description

This project is a simple HTML5 webpage designed to demonstrate the use of semantic HTML elements along with basic CSS styling. It includes sections like header, footer, aside, and figure, along with embedded multimedia content.

🚀 Features
Uses HTML5 semantic tags (header, section, footer, aside, figure)
Simple and clean layout using Flexbox
Embedded YouTube video
Embedded Google Map
Basic CSS styling for layout and design
🛠️ Technologies Used
HTML5
CSS3
📂 Structure
Header: Displays the page title
Section: Main content area
First row: Text-based aside and figure
Second row: Embedded video and map
Footer: Copyright information
▶️ How to Run
Copy the code into a .html file (e.g., index.html)
Open the file in any web browser
📷 Output Overview
Top header with title
Two flexible content rows
Embedded YouTube video
Embedded Google Map
Footer at the bottom
